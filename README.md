**DICCIONARIO**
el diccionario
es una conexion de datso que los almacena en 
partes clave valor 

- un diccionario comienza y termina con llaves 
(curly braces) 
- la clave se separa del avlor por dos puntos 
- cada elemento o propiedad del diccionario se 
separa por una coma.

**EJEMPLO**

diccionario que almacene los datos de un 
pais
